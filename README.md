## 📖 Project Description

The **Hotel Njenka Website** is a responsive web application developed to showcase hotel services, rooms, and booking information in a modern and user-friendly way.

This project was created as part of my studies at **Savonia University of Applied Sciences**, focusing on web development, responsive design, and basic JavaScript functionality.

The website provides visitors with information about the hotel, available rooms, services, and contact details.

---

## 🎯 Project Objectives

The main goals of this project were to:

* Design a modern hotel website interface
* Practice HTML, CSS, and JavaScript skills
* Build a responsive website that works on different devices
* Create an organized website structure
* Improve user experience with simple interactions

---

## 🛠️ Technologies Used

The following technologies were used to build this project:
